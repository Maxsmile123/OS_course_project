### Description

The application is a Web messenger in an initially dark theme (see the layout below).
To write to someone, you need to poke at him in the list of users -> his name to move to the lower right block showing
who we have an active chat with. The button in the upper left corner pushes the menu.

### Functional

- Sending a message to all messenger users :white_check_mark:
- Sending private messages :white_check_mark:
- Authorization and registration :white_check_mark:
- List of online/offline users :white_check_mark:
- Ability to change the name :white_check_mark:

### Possible improvements:
